Preguntas Frecuentes

¿Necesito un servidor especial? +

No. El generador funciona en cualquier servidor web. Puedes alojarlo en Apache, Nginx, GitHub Pages, o incluso usarlo local sin servidor.

¿Mis preguntas se almacenan en la nube? +

No. Todo ocurre en tu navegador. Las preguntas nunca salen de tu computadora. El generador no tiene backend ni base de datos.

¿Cuántos tests puedo generar? +

Ilimitados. Genera 1, 5, 10 o 100 tests. El generador funciona igual sin importar cuántos crees.

¿Puedo personalizar el diseño de los tests? +

Sí. Los HTMLs generados son código completo. Puedes editar colores, fuentes, textos y comportamiento como necesites.

¿Funciona offline? +

El generador sí funciona offline. Los tests generados funcionan offline también (excepto la descarga inicial del ZIP que requiere navegador).

¿Qué formato debe tener mi archivo .txt? +

Consulta la documentación ESTRUCTURA_ARCHIVOS.md para ver el formato exacto. El parser es flexible y tolera variaciones menores.

¿Puedo integrar esto en mi aplicación? +

Sí. El código es tuyo. Puedes extraer el JavaScript, usarlo como iframe, o adaptarlo como necesites.

¿Hay soporte técnico? +

La documentación incluye CHEATSHEET.txt para referencia rápida, ESTRUCTURA_ARCHIVOS.md para el formato, e INSTRUCCIONES_DESPLIEGUE.md para hosting.
